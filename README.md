# LinkedList

- Linear data struct.
- Elements are stored in non contiguous locations
- Elements are linked using pointers

## ADVANTAGE OF LL:
- Dynamic size
- Ease of insertion and deletion

## DISADVANTAGE OF LL:
- Random access is not allowed
- Extra memory for pointers
- Not cache friendly

## Complexites:
insertion/deletion -- O(1)
Access element -- O(n)
